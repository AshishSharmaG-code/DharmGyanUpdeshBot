DharmGyanUpdesh Telegram Bot - Setup Instructions (Mobile Friendly)

1. Telegram Bot Token:
   - Open Telegram app
   - Search @BotFather
   - Create a new bot with /newbot and get the token
   - Replace 'YOUR_BOT_TOKEN_HERE' in bot.py with your actual token

2. Run Locally (Optional):
   - Install python-telegram-bot library using:
     pip install -r requirements.txt
   - Run bot.py using Python

3. Deploy on Render.com for 24/7 Hosting:
   - Create a GitHub repository and upload bot.py and requirements.txt
   - Sign up/login to https://render.com
   - Create a new Web Service linked to your GitHub repo
   - Set Environment to Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: python bot.py
   - Deploy and get your live URL

4. Use Your Bot:
   - Open Telegram
   - Search your bot username
   - Type /start to begin
   - Type /gyan to get spiritual quotes

5. Notes:
   - Keep your bot token secure
   - Update gyan_list in bot.py to add more messages
